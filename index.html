Appointments - 2023

Projeto de código aberto que consiste em uma API REST para gerenciamento de compromissos. Desenvolvida usando Node.js, Express.js, JWT, TypeORM e Typescript. Possui sistema de validação e autenticação com diferentes cargos. Um email de lembrete é enviado 24 horas antes de cada compromisso através de um Cron Job que é executado em intervalos de 10 minutos. Todos os endpoints da API são testados utilizando Jest.

Library - 2023

Sistema de gerenciamento de biblioteca que consiste em uma API REST desenvolvida em Laravel 11 e consumida com Vue.js 3. O sistema contém gerenciamento de usuários, livros, gêneros e empréstimos. Todos os endpoints da API são testando com PHPUnit, além de ter o design pattern repository implementado para manter o código mais limpo, organizado e legível. Essa organização acontece pois esse design pattern cria uma camada a mais de abstração entre o controller e model.

Link para o repositório do projeto, contribua.

Laravel Vue.js Bootstrap MySQL
Notes - 2021

Sistema de controle de notas/lembretes. Crie, gerencie e exporte suas notas em um sistema muito completo. Utilizei Laravel para a API REST com autenticação JWT e por sessão no back-end e Vue.js para deixar o front-end mais dinâmico em conjunto com Bootstrap. Contém recurso de exportar notas para Excel e PDF, registro de logs em todos os endpoints da API de notas e sistema de login completo. Também implementei Design Pattern Repository e testei todos os endpoints da API utilizando PHPUnit. Código fonte disponível no repositório do projeto, toda contribuição é bem-vinda.

Link para o repositório do projeto, contribua.
